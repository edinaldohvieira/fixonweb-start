Redireciona conforme a função a que o usuário pertence.

== Description ==

Muito util para redirecionar imediatamente após o usuario se logar.

Um exemplo:
Se o usuario tiver a função de "administrativo" então será redirecionado para a página /administrativo.
Se o usuario tiver a função de "cliente" então será redirecionado para a página /minhas-compras/.


== Instalação ==

This section describes how to install the plugin and get it working.

e.g.

1. Instale o plugin automaticamente através do Painel do WordPress ou fazendo upload do arquivo ZIP no diretório de plugins.
1. Depois, após o upload e a extração do pacote, clique em Ativar Plug-in.
